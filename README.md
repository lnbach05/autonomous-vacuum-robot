# Autonomous Vacuum Robot

Advanced autonomous robotic vacuum cleaner platform built using:

- ESP32
- Raspberry Pi 5
- LiDAR
- ROS2
- SLAM
- Embedded motor control
- Autonomous navigation

---

# Project Goals

- Autonomous cleaning
- Real-time room mapping
- Intelligent obstacle avoidance
- Efficient coverage path planning
- Future docking station support

---

# Repository Structure

firmware/   -> ESP32 embedded firmware
software/   -> ROS2 and high-level autonomy
hardware/   -> CAD, wiring, PCB files
docs/       -> engineering documentation
testing/    -> subsystem validation
media/      -> photos/videos/renders
