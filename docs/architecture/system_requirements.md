# System Requirements

## Mechanical
- Diameter: 15 inches
- Height: <5 inches
- Weight target: <10 lbs

## Navigation
- Autonomous mapping
- Wall following
- Obstacle avoidance
- Cliff detection

## Cleaning
- Vacuum suction
- Side brushes
- Roller brush

## Sensors
- LiDAR
- Ultrasonic
- IMU
- Encoders

## Compute
- ESP32 real-time controller
- Raspberry Pi high-level autonomy

## Battery
- Minimum 45 minutes runtime

## Software
- ROS2
- SLAM
- PID motor control
